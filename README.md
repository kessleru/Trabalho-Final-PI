# Pokédex

Pokédex web feita com React + Vite, consumindo a [PokeAPI](https://pokeapi.co/).

## Funcionalidades

- Listagem de Pokémon com paginação
- Filtro por tipo
- Modal com detalhes, stats e habilidades

## Como rodar

```bash
pnpm install
pnpm dev
```

## Tecnologias

- React
- Vite
- Lucide React (ícones)
- PokeAPI
