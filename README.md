# Pokédex

Aplicação web de Pokédex para explorar e consultar Pokémon de forma rápida, com listagem paginada, filtro por tipo e modal de detalhes.
O projeto foi construído com React + Vite e consome a [PokeAPI](https://pokeapi.co/).

## Pré-requisitos

- Node.js instalado
- `pnpm` instalado

## Funcionalidades

- Listagem de Pokémon com paginação
- Filtro por tipo
- Modal com detalhes, stats e habilidades

## Instalação

Instale as dependências do projeto com:

```bash
pnpm install
```

Se preferir usar `npm`, execute:

```bash
npm install
```

## Como rodar localmente

Inicie a aplicação em modo de desenvolvimento com:

```bash
pnpm dev
```

Com `npm`, use:

```bash
npm run dev
```

Depois, abra o endereço exibido no terminal no navegador.

## Tecnologias

- React
- Vite
- Lucide React (ícones)
- PokeAPI
